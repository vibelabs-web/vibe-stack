My First React Component!

A breakdown of how I finally understood props and state logic after struggling for a week.

Sarah Jenkins Beginner
Posted 2 hours ago â€¢ 5 min read

I've been learning web development for about 3 months now, and I hit a wall when I started learning React. The concept of writing HTML inside JavaScript (JSX) felt weird, and props just didn't make sense to me.

But today, it finally clicked! I built a simple counter app, but instead of putting everything in one file, I split the button into its own component.

The Breakthrough

Here is the code for my custom button component. I realized that props are just arguments you pass to a function!

CustomButton.jsx 
function CustomButton({ onClick, label }) {
  return (
    <button 
      className="bg-blue-500 text-white px-4 py-2 rounded"
      onClick={onClick}
    >
      {label}
    </button>
  );
}

export default CustomButton;
